repo toepassen**.

## Hoe gebruik je dit?

- **Ontwerpers/consultants:** loop bij [stap 06–09](../blueprint/) de relevante pagina
  langs; gebruik de [alignment-matrix](microsoft-alignment.md) als checklist.
- **Onderhoud:** werk de bronlinks + "laatst getoetst"-datum bij als Microsoft de
  guidance wijzigt (zie de [What's new](https://learn.microsoft.com/microsoft-copilot-studio/guidance/whats-new)).

> Voorbeelden rond regelgeving (Bbl/NEN) blijven illustratief; laat regelgeving door
> een bevoegd persoon toetsen.
